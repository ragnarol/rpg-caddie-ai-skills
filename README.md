# rpg-caddie-ai-skills

Custom [Claude Cowork](https://claude.ai) skills for the *Pirates of Drinax* Traveller RPG campaign.

## Skills

| Skill | Description |
|---|---|
| `game-session` | Process a session summary through the full vault-update workflow |
| `dedupe-npcs` | Scan for duplicate NPC files, merge, and remove redundant copies |
| `fix-npc-last-seen` | Audit and correct `last_seen` / `place_last_seen` metadata across all NPC files |
| `extract-ship-images` | Extract ship illustrations from Traveller RPG PDFs as named PNGs |
| `foundry-npc` | Create and populate a Traveller NPC actor in Foundry VTT (MGT2 system) |
| `foundry-ship` | Create and populate a Ship actor in Foundry VTT (MGT2 system) |
| `update-foundry-portrait` | Update a Foundry VTT actor portrait and token from the Obsidian vault |
